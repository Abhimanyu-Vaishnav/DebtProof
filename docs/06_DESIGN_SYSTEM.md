# 06. Design System

This document captures the color palette, typographical weights, and metrics defining the visual language of the DebtProof brand.

## Colors & Palette

| Token Name | HEX Value | Purpose / Usage |
| :--- | :--- | :--- |
| **Primary Navy** | `#17375E` | Navigation bars, primary buttons, headers |
| **Primary Blue** | `#214E83` | Action icons, active tab highlights |
| **Background** | `#F5F7FB` | Main viewport canvas, body background |
| **Cards** | `#FFFFFF` | Form boxes, data card surfaces |
| **Borders** | `#E5EAF2` | Grid separators, form input borders |
| **Primary Text** | `#1B2430` | High-contrast copy, headers, bold text |
| **Secondary Text** | `#7B8BA5` | Captions, subtitle strings, descriptions |
| **Success** | `#2CCB7F` | Verified proofs, closed status, check icons |
| **Warning** | `#F6A623` | Pending items, warning banners |
| **Danger** | `#EF5350` | Failed items, deletion controls, errors |
| **Purple Accent** | `#6C4CF1` | Monad branding accents, Web3 interactions |

---

## Typography
- **Primary Font Family**: Inter, sans-serif
- **Sizes**:
  - `Display Title`: `32px` (Bold)
  - `Header Title`: `20px` (Bold)
  - `Sub-header`: `16px` (Semi-bold)
  - `Body Copy`: `14px` (Regular)
  - `Small Caption`: `12px` (Medium/Regular)
  - `Tiny Detail`: `10px` (Bold)

---

## Layout Metrics
- **Base Grid Unit**: `8px`
- **Border Radius**:
  - Small elements (Badges): `4px`
  - Medium elements (Inputs): `12px`
  - Large elements (Cards): `24px`
- **Padding & Margins**:
  - Compact: `12px`
  - Normal Page: `20px`
  - Large Spacing: `32px`
- **Elevation / Shadow**:
  - Card Shadow: `BoxShadow(color: Color(0x05000000), blurRadius: 10, offset: Offset(0, 4))`
